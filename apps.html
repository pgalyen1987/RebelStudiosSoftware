Indemnity (INDN)

An ERC-20 token where ordinary transfers are held in on-chain escrow for 7 days instead of settling instantly. During that window the payer can open a dispute and a decentralized Kleros jury decides who receives the funds. DEX swaps settle instantly via an owner-managed exemption allowlist.

  • Kleros decentralized jury for dispute resolution
  • DEX exemption allowlist for instant swaps
  • Stuck-dispute escape hatch (180-day timeout)

Vigilo

An open-source research prototype that learns normal device behavior from Zeek connection logs and flags anomalies as forecast error. Detects port scans, DDoS, and noisy botnet traffic on consumer CPU hardware with no cloud dependency.

Technical Implementation

Core
Python Mamba-2 SSM PyTorch Zeek conn.log
Detection

Key Features

  • 5-minute behavioral windows per device (15 features)
  • 75% detection at 1% FPR on IoT-23 (20 malware families)
  • Runs entirely on CPU — no GPU or cloud required
  • SPAN/mirror, gateway Zeek, or lab capture workflows
